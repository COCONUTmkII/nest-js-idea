Detects NestJS controllers that are not provided in any module.

This inspection helps identify controllers that are decorated with @Controller but are not listed in the controllers array of any NestJS module. Such controllers will not be instantiated by the framework and may cause runtime errors.

Example:


import { Controller } from '@nestjs/common';

@Controller()
export class MyController {
  // ...
}
// If MyController is not added to any module's controllers array, it will be flagged by this inspection.

To fix this, ensure the controller is included in the controllers array of a module, e.g.:


import { Module } from '@nestjs/common';
import { MyController } from './my.controller';

@Module({
  controllers: [MyController],
})
export class MyModule {}
